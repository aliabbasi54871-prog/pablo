# NARCOS — Storytelling Landing

Документальный storytelling-лендинг для проекта «История и память в цифровую эпоху».

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

## Структура истории (10 глав)

1. **Hero** — Medellin + Escobar 1989, parallax zoom
2. **Город до легенды** — горизонтальный storytelling
3. **Возникновение Эскобара** — вертикальный таймлайн + подпись
4. **Пик могущества** — 80% / 30 млрд $
5. **Нарковойна** — стиль расследования
6. **Конец Эскобара** — полноэкранная драма
7. **Память** — могила, борьба за нарратив
8. **Медельин сегодня** — split-screen 1980-е ↔ 2020-е
9. **История или миф?** — интерактивная сеть
10. **Финал** — вывод + источники
